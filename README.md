# Attendance Management System

WPF-based attendance management system for students and teachers (C#, .NET).

## Tech
- .NET (WPF)
- C#
- Entity Framework

## Quick Start
1. Open `AttendenceManagementSystem.sln` in Visual Studio.
2. Build the solution.
3. Run the application.
